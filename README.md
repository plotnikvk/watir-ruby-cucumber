# watir-cucumber-ruby

## Getting Started

      1. Ensure you have the chrome browser installed
      3. Ensure you have ruby + devkit installed. File in https://rubyinstaller.org/downloads/
      4. Ensure you have rubygems
      5. Write in concole :
         - gem install bundler
         - bundle install
      5. Run s.bat

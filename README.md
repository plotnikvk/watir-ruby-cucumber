# watir-cucumber-ruby

## Getting Started

      1. Ensure you have the chrome browser installed
      3. Ensure you have ruby installed. File in 'file_for_work'
      4. Ensure you have rubygems
      5. Write in concole :
      gem install bundler
      bundler install
      5. Run 'bundle install'
      6. Run 'cucumber'
